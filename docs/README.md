# 📚 Documentation & Research

KriptoSancak projesine ait teknik makaleler, mühendislik notları ve mimari diyagramların bulunduğu ana arşivdir.

## 📁 Yapı
- **Technical Papers:** Kriptografik ispatlar ve algoritmik analizler.
- **RFC:** Proje içi standart teklifleri.
- **Guides:** Kurulum ve güvenli kullanım kılavuzları.

## ✍️ Katkı
Yeni bir teknik makale veya döküman eklemeden önce lütfen `RFC` sürecini takip edin.
