# 🌌 Post-Quantum Cryptography (PQC)

Kuantum bilgisayarların mevcut asimetrik şifreleme yöntemlerini (RSA, ECC) tehdit ettiği bir geleceğe hazırlık amacıyla oluşturulmuş araştırma ve geliştirme dizinidir.

## 🧪 Araştırma Alanları
- **Lattice-based Cryptography:** Kyber, Dilithium gibi NIST standart adayları.
- **Isogeny-based Cryptography:** SIDH (Araştırma amaçlı).
- **Code-based Cryptography:** McEliece kriptosistemi.

## 🎯 Hedef
"Q-Day" gelmeden önce sistemlerin kuantum dirençli algoritmalara sorunsuz geçişini sağlayacak kütüphaneleri optimize etmektir.
